# Backup Artifacts Mirror

This directory stores Git-tracked backup artifacts in a rolling window.

Layout:

1. `runs/<timestamp>/ops.bundle`: Git bundle copied from `/opt/backup/git-bundles`
2. `runs/<timestamp>/data.tar.gz`: data snapshot copied from `/opt/backup/data-snapshots`
3. `runs/<timestamp>/backup-meta.env`: backup timestamp, source files, and SHA-256 checksums
4. `latest/*`: symlink pointers to the newest run

Retention behavior:

1. `backup.sh` keeps only the latest `N` run directories in Git working tree (`N=3` by default).
2. Old run directories are removed from Git history by `scripts/prune-backup-history.sh` (requires clean tracked worktree).
3. Override keep count with `--repo-artifacts-keep <count>`.
4. Skip history rewrite with `--skip-repo-history-prune` (not recommended for long-term repo size).

Generated by:

1. `bash /opt/ops/scripts/backup.sh`
